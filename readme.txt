== Description ==

This plugin will help you to create product specific frequently bought together products.

== DISCOUNT TYPES AND FEATURES ==

* Site admin can create product specific frequently bought together products for the products. It will helps to improve products sales. You can set frequently bought together products for every invidual product.Customer can add to cart the frequently bought together products on the product page. You can set multiple frequently bought together products for one product.

== Installation ==

Download the plugin in your directory. Upload it in your site add plugin page and active the plugin. Also you can manually copy the plugin folder in your site plugin folder PATH(your_wordpress_path/wp-content/plugins) and you can active the plugin.  

= Minimum Requirements =

* WordPress 4.0.0 or greater
* WooCommerce 3.0.0 or greater
* PHP version 5.6.0 or greater
* MySQL version 5.0 or greater
